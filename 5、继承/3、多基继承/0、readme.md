## 多基继承（多基派生）

### 1. 多基继承的定义形式
```
class 派生类
:public/protected/private 基类1
, //...
,public/protected/private 基类n
{

};
```

### 2. 多基继承派生类对象的构造和销毁
1. 派生类构造函数的格式
```
派生类名(总参数列表)
:基类1 (参数列表1)
,基类2(参数列表2)
, //...
,基类n(参数列表n)
{
    //函数体
}
```

2. 成员名冲突的二义性
> 解决方式：只要在调用时写明调用的是某个基类的成员函数（作用域限定符）
