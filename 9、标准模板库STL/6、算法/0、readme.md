### 分类
1. 非修改式序列操作：不改变容器内容
2. 修改式序列操作：可以改变容器中的内容
3. 排序和相关操作：包块各种排序函数
4. 通用数字运算：计算两个容器的内部乘积等

### 一元函数及一元断言、一元谓词
一元函数：函数的参数只有一个
一元断言、一元谓词：函数的参数只有一个，并且返回类型是bool类型
二元函数：函数的参数有两个
二元断言、二元谓词：函数的参数两个，并且返回类型是bool类型
